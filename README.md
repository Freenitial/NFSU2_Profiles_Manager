# Need For Speed Underground 2 Profiles Manager

A simple tool to manage profiles and pseudo for NFSU2

---

## Features :
- **Create** pseudo (preconfig : XBOX 360 controller, AZERTY keyboard)
- **Modify** pseudo
- **Clone** profile into a new pseudo
- **Delete** profile
